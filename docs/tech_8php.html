Detailed Description

Page dédiée aux développeurs, présentant des interactions avec des APIs :

Author
Lisa/Ouardia
  • API NASA Astronomy Picture of the Day (APOD)
  • API de géolocalisation via JSON et XML
  • API whatismyip.com pour la localisation IP détaillée

Cette page récupère l'IP du visiteur, ses informations de géolocalisation en formats JSON et XML, l’image ou la vidéo du jour de la NASA, ainsi que des infos avancées via un service IP externe.

PHP version 8.1+
